# Fluxnova Platform Engine Quarkus Extension

You can add the Fluxnova Platform Engine Quarkus Extension into your Quarkus application by adding
the following dependency into your `pom.xml` file.

```xml
<dependency>
  <dependency>
    <groupId>org.finos.fluxnova.bpm.quarkus</groupId>
    <artifactId>fluxnova-bpm-quarkus-engine</artifactId>
    <version>${version.fluxnova}</version><!-- place Fluxnova version here -->
  </dependency>
</dependency>
```
